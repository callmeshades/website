The application allows personal trainers to create programs to assign to their clients.
Originally created in Vue but migrated to React. The backend is written in Django. At the time of creating the application, I went for a NoSQL approach.
If I was to recreate this now, I would 100% use a relational database to make managing records much easier. It would keep everything properly in sync.
