Single page application (SPA) with a Django backend to allow foremen to fill out daily reports on site. HR can download reports as a PDF back at the office. The application can be used on tablets, mobile, and desktops. This is in use daily. I chose Vue as a technology to be able to dynamically alter a table with ease and react to new data.
