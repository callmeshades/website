Created a desktop application for scraping Nivel. 
This was my first time creating a complete application in Go. 
Go was required as there are hundreds of thousands of products to scrape and using it allowed them to be scraped in a timely manner. 
This included authentication for their account to get discounted prices. The GUI was created using Fyne. Goroutines were used for scraping mass amounts.
